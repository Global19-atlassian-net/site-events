Practical information

What is the Dart Developer Summit? The Dart Developer Summit is an event that connects Dart developers to each other and to the engineers on the Dart team. In two days, you'll learn the latest about Dart's future and about the practicalities of using Dart to build apps. The Summit is single tracked with short presentations to ensure everyone is part of an engaging conversation.
